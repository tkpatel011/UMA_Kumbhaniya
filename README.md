# 🌸 Uma Kumbhaniya

## 🏬 About

Uma Kumbhaniya is a clothing and saree-based business that offers traditional and modern fashion with elegance and quality. The aim is to provide customers with stylish, comfortable, and affordable ethnic wear.

---

## ✨ Features

* 🛍️ Wide variety of sarees and traditional wear
* 📍 Shop information and location
* 🖼️ Image gallery of products
* 📄 Downloadable menu/catalog
* 📱 Fully responsive website (mobile + desktop friendly)

---

## 🌐 Website Overview

This project is an informative website designed to:

* Show shop details
* Display product collections
* Help users find the shop easily with a mini map
* Allow users to view and download the menu

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* (Optional: Flutter for mobile app integration)
